Configuring Columns to sync

[FUTURE ENHANCEMENT] If there are certain columns from Smartsheet that you do not want synced over, you can tweak the JSON config file accordingly before actually syncing the data.

Open the config-[SheetID].json file in the text editor of your choice. For any columns you do not want synced over, change the Sync property from true to false
