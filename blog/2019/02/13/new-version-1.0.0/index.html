New Version 1.0.0

February 13, 2019

Alan Westley

We are happy to announce the release of v1.0 of the SmartsheetToSQL PowerShell tools. This initial release allows you to sync data from Smartsheet to a SQL Server database. Please read the docs for additional information.

We are looking to expand our Smartsheet PowerShell offerings. If there is additional functionality that you are interested in seeing, please email us at contact@inflectionit.com
